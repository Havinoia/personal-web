# Havin Portfolio — Pure Black Elegant 🌑✨

**Havinoia Portfolio** is a high-end, modern web application designed to showcase a professional journey in Fullstack Development. Focused on a "Pure Black Elegant" aesthetic, this portfolio blends high-performance code with state-of-the-art UI/UX design.

---

## 🔥 Overview
This project is more than just a resume—it's a curated experience. It utilizes cutting-edge web technologies to create a fast, interactive, and visually stunning demonstration of development skills. From glassmorphism to neon-green highlights, every pixel is crafted to leave a lasting impression.

## 🌟 Key Features
- **Modern UI/UX**: A bespoke design system using HSL color tokens, smooth gradients, and glassmorphism cards.
- **Interactive Section Scrolling**: Custom scroll behaviors that handle cross-page navigation to specific anchors (Portfolio, Tech Stack, etc.) with ease.
- **Dynamic Project Carousel**: A high-performance, looped project showcase with support for drag-and-drop and momentum scrolling.
- **Immersive Micro-interactions**: 
  - **3D Tilt Effect**: Tilt animations on certification cards that respond to mouse movement.
  - **Dynamic Typewriter**: A smooth hero typewriter effect for multiple roles.
  - **Scroll Reveal**: Sections fade and slide into view for a more organic reading experience.
- **Modular Architecture**: Built with a "Store-First" approach using Pinia, allowing for easy updates to projects, skills, and profile data.

## 🛠️ Core Tech Stack
- **Frontend**: [Vue 3](https://vuejs.org/) (Composition API / Script Setup)
- **State Management**: [Pinia](https://pinia.vuejs.org/)
- **Routing**: [Vue Router 4](https://router.vuejs.org/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/) & Vanilla CSS for custom animations.
- **Integration**: EmailJS for real-time contact form handling.
- **Bundler**: [Vite](https://vitejs.dev/) for lightning-fast build and development.

## 📂 Project Highlights
The portfolio features several sophisticated projects, including:
- **TypeRush**: A high-speed typing engine with real-time WPM tracking.
- **Nusantara Logistics**: A comprehensive TMS with interactive fleet mapping.
- **VinShop**: A full-featured e-commerce platform with payment gateway integration.
- **HealthTic Monitor**: A real-time data visualization desktop utility.

---

### 🎨 Design Philosophy
The design follows a **Pure Black** theme, prioritized for high-end monitors and OLED screens. It uses Neon Green accents (`#7dff9b`) to guide the user's focus, creating a professional yet futuristic atmosphere that aligns with modern development standards.
